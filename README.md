# KLinika

## Project info

The Clinic Management System is a web application developed using Node.js, React, and MySQL, which enables complete management of administrative and medical processes in a clinic. The system provides secure authentication and authorization with different roles (Admin, Doctor, Staff) using JWT, as well as management of patients, doctors, and medical services through full CRUD operations. It allows appointment scheduling and management, payment recording, storage of patient medical history, and generation of statistics through an interactive dashboard with charts and analytics data. The database structure is organized in a relational manner to ensure data integrity and security, while the application is optimized for performance and works on major web browsers.
